# House price Prediction using Linear Regression (Kaggle Dataset)
# linear-regression-kaggle

This project applies Linear Regression to predict an employee's Salary based on their Years of Experience.

# Dataset Information
- File name: `Housing.csv`
- Rows: 546
- Columns: 2

# Columns:
- price → Input Feature (X)
- area → Target Variable (y)

## Objective
To build a Linear Regression model that learns the relationship between price and area, and predicts house price for new house.

## Library Used
- Pandas
- NumPy
- Matplotlib
- Scikit-learn# Linear-Regression
